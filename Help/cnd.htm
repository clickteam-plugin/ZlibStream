Conditions

The ZlibStream object has 2 conditions.

On success
On error

On success:
On error is TRUE when a compression or a decompression action finishes. Use the LastOutputFile$("ZlibStream") expression to get the last completed compression's or decompression's output file.
Parameters: None.

On error:
This condition is TRUE when an error has occured. This can be caused by a compression or decompression being unable to start properly, or by trying to use an invalid buffer size. Use the ReturnedString$("ZlibStream") expression to get a description of the error.
Parameters: None.
